## Installation ğŸ› ï¸

1. Clone the repository:

   ```
    git clone https://github.com/NikhilKalloli/Blog.git
   ```

2. Navigate to the project directory:
    ```
    cd Blog
    ```
3. Install dependencies:
    ```
    npm install
    ```

    ```
    node index.js
    ```
